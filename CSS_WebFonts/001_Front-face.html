Текст данного элемента написан шрифтом heinrich.

В предыдущих версиях CSS разработчики были вынуждены использовать только те грифты, которые гарантированно установлены на компьютере пользователя, в CSS3 разработчики могут использовать любые шрифты, которые они захотят. Когда вы найдете необходимый шрифт, просто разместите его на веб-сервере и подключите его с помощью CSS3 свойства @font-face. Подключённый грифт будет скачан и отображён автоматически при загрузке страницы.

Обратите внимение: Internet Explorer поддерживает шрифты в формету .eot, а остальные браузеры в формате .ttf т.е. для того, чтобы шрифт отображался корректно во всех браузерах Вы должны подключить его к странице в двух форматах.
